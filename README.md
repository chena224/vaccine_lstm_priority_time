# 基于LSTM与双注意力机制的疫苗效果时序预测模型

本项目针对疫苗免疫效果预测任务，构建融合**优先级注意力**与**时间注意力**的LSTM模型，实现对疫苗第2月、第6月阳转率的精准回归预测。
模型结构完整、可解释性强、工程规范，适合科研展示与算法岗位简历项目。

## 项目特点
- 双注意力机制：优先级注意力（疫苗成分/疾病/人群）+ 时间注意力（历史阳转率）
- LSTM时序建模 + 初始嵌入H0引导，提升时序特征捕捉能力
- 完整训练流程：早停、梯度裁剪、余弦退火学习率、AdamW优化器
- 全维度可视化：训练曲线、预测对比图、注意力权重热力图
- 数据校验严格：样本ID对齐、维度校验、数值范围约束，保证实验可复现

## 任务说明
- 任务类型：多输出回归预测（第2月、第6月疫苗阳转率）
- 输入特征：3类优先级32维嵌入 + 初始状态H0 + 历史时间特征Y0
- 模型结构：LSTM + 优先级-时间双注意力 + LayerNorm + Dropout
